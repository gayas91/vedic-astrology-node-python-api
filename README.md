# Vedic Astrology Node.js + Python API Engine

Production-grade Vedic Astrology backend powered by Node.js, Python, and Swiss Ephemeris, supporting Panchang, Horoscope, Matchmaking, Dasha Systems, KP Astrology, Ashtakvarga, Shadbala, Bhava Bala, Festival Calendar, and Divisional Chart Calculations.

## Overview

A production-ready Vedic Astrology backend engine built using Node.js and Python.

The system follows a hybrid architecture where Node.js manages API routing and business logic, while Python performs astrology calculations using Swiss Ephemeris.

### Architecture

Client Application
        │
        ▼
Node.js REST APIs
        │
        ▼
Business Logic Layer
        │
        ▼
Python Astrology Engine
        │
        ▼
Swiss Ephemeris
        │
        ▼
Structured JSON Response

## Installation

```bash

cd vedic-astrology-node-python-api

python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt

npm start
```
